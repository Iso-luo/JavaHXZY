内部类：

class A{

	class B extends C {}

}


内部类的对象要依托外部类来存在
----------------------------------
内部类：

	内部类当做成员属性存在( 成员内部类 )

		1,内部类可以无限制的访问外部类的私有属性
		2，内部类可以当做一个独立的类 来继承其他的类
	内部类在成员方法中(  局部内部类   )

	
	注意：内部类与外部类不存在继承关系
	
	
匿名内部类	
	匿名对象
		可以当做参数进行传递
		节约内存资源
	内存类
		节约内存资源
		
-------------

代码块:

	1, 优先于 构造函数运行
	2， static 代码 ，在对象创建之前 只能执行1次
